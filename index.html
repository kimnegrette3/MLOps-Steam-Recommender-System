Steam API Documentation

Root

Endpoint para obtener información general sobre la API.

Para acceder a esta página, utiliza la siguiente URL:

/

User Data

Endpoint para obtener datos de usuario.

User Data: Recibe un string con el "User Id" y devuelve un summary del user Se puede acceder desde esta URL: 

https://steam-api-project.onrender.com/userdata/{user_id}

Ejemplo de uso: https://steam-api-project.onrender.com/userdata/--000-- Resultado: { "user_id": "--000--", "procentaje_recomendacion": 100, "cantidad_gastada": 402.77, "cantidad_items": 58 }

Sentiment Analysis

Endpoint para realizar análisis de sentimientos basado en un año.

Para acceder a este endpoint, utiliza la siguiente URL:

/sentimentanalysis/{year}

Ejemplo de uso:

/sentimentanalysis/2010

Genre

Endpoint para obtener información sobre un género específico.

Para acceder a este endpoint, utiliza la siguiente URL:

/genre/{genre}

Ejemplo de uso:

/genre/RPG

User for Genre

Endpoint para obtener una lista de los usuarios con más horas de juego en un género.

Para acceder a este endpoint, utiliza la siguiente URL:

/userforgenre/{genre}

Ejemplo de uso:

/userforgenre/Indie

Developer

Endpoint para obtener información sobre un desarrollador específico.

Para acceder a este endpoint, utiliza la siguiente URL:

/developer/{developer}

Ejemplo de uso:

/developer/Mortis Games

Game Recommendations

Endpoint para obtener recomendaciones de juegos basado en un juego específico.

Para acceder a este endpoint, utiliza la siguiente URL:

/game_recommendations/{game_id}

Ejemplo de uso:

/game_recommendations/123
